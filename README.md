# Singapore Property Search Skill

Search Singapore properties on 99.co. Find condos, HDBs, or landed properties for rent or sale, filter by MRT, district, price, or size.

## Installation

```bash
npx add-skill yemyat/singapore-property-search-skill
```

## Example Prompts

- "Find me HDBs near Bugis area within 5-10 mins walking distance from the MRT"
- "Show me 3-bedroom condos for rent in Orchard under $5000/month"
- "Search for landed properties in District 10 for sale"
- "Find pet-friendly rentals near Tampines MRT"
